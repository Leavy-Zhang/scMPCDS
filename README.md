## Major Steps

0) To activate environment: 

	conda activate scMPCDS

*1) Go to the directory with `01.RawData/` containing raw sequeincing data, `ref_base/`containing mitochondria genome fasta file, gene-anntation gtf file and the GRCh38 genomic single cell ATAC refdata downloaded from 10X webpage, `scMPDCS/` containing these scripts:
   
    cd /path/to/my_scmpcds_project
   
*2) To obtain vcf files with high quality for each cell. please run scMPCDS/seq2vcf/mapping2vcf.sh:

    ./scMPCDS/seq2vcf/mapping2vcf.sh

*4) merge all vcf files generated by step 2):
   
    Rscript  scMPCDS/seq2vcf/mergeVCFs.R

*5) With the generated vcf-merged csv table, you can interactively perform analysis following the R scripts scMPCDS/mutation_statistics/Statistics_plot.r step by step.

*6) To perform triNucleotide analysis, just run:
   
	python scMPCDS/scMPCDS_QC/triNucAnalysis.py

*7) To perform the simulation analysis of the growth of nucleotide error rate during PCR duplication, please run:
   
	python scMPCDS/scMPCDS_QC/pcr_error_simulation.py

*8) To perform the basic 10X scATAC-seq analysis, please run:

    ./scMPCDS/10X/fastq2mgatk.sh
	
## Directories

**seq2vcf/**: contains shell/R scripts how to generate the basic SNP table using scMPCDS raw sequencing data

**inputExample/**: contains several merged SNP .csv table files generated by steps contained in **seq2vcf/**

**scMPCDS_QC/**: contains python scripts how to perform related machine learning, simulation and triNucleotide analysis.

**10X/**: contains shell/R scripts how to perform mitochondria SNP allel analysis using 10X ATAC-seq data

**mutation_statistics**: contains an integrated R script, containing all details about rest statistical analysis and data visualization.

**Figures/**: contains all figures generated through Bioinformatic Machine learning analysis
